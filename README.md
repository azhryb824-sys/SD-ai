# SD-ai

مساعد عربي بواجهة وهوية سودانية، يدعم:

- المحادثة بالدارجية السودانية.
- البحث والتحليل والمقارنة.
- الردود النصية والصوتية.
- استنساخ الصوت محلياً عند توفر ملفات الصوت والنموذج.
- تنويع العبارات الترحيبية السودانية.

## التشغيل

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
uvicorn app:app --host 127.0.0.1 --port 8001
```

ثم افتح:

```text
http://127.0.0.1:8001/
```

## ملفات الصوت

التسجيلات الصوتية الخام، الملفات المولدة، والكاش المحلي غير مرفوعة إلى
المستودع حفاظاً على الخصوصية وتقليل الحجم. يمكن إضافتها محلياً داخل
`voice_samples` وتشغيل أدوات التحضير الموجودة في المجلد نفسه.

## النشر على Render

المشروع يحتوي على ملف `render.yaml` جاهز للنشر كخدمة ويب. اربط مستودع
GitHub مع Render واختر **New Blueprint Instance**؛ سيستخدم Render إعدادات
البناء والتشغيل تلقائياً.

نسخة Render تشغّل الدردشة النصية والبحث والتحليل بمتطلبات خفيفة. استنساخ
الصوت المحلي غير مفعّل في النشر السحابي لأن ملفات الصوت والنموذج كبيرة
وخاصة، وتحتاج موارد أعلى من الخطة المجانية.
